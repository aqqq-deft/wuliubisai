# CODBOT-RCU 底盘串口调试工具

## 使用前

- 底盘必须稳定架空，首次测试不得让轮子接触地面。
- 先确认四个电机和编码器接入主板 A/B/C/D 端口。
- 选择名称包含 `USB-SERIAL CH340` 的端口，不要选择蓝牙 COM3/COM4。
- STM32必须烧录正式控制固件；RTX诊断固件不会响应电机命令。

## 使用步骤

1. 双击 `启动底盘调试工具.cmd`，无需安装Python。
2. 刷新串口，选择CH340端口并连接。
3. 保持最低速度100 mm/s、0.30 rad/s。
4. 按住按钮测试，松开确认立即停车；异常时点击红色急停并断开电池。
5. 顺序测试前进/后退、横移、原地转向、前进转向，首次每次约1秒。
6. 测试通信保护：运动时停止发帧或拔掉Type-C，必须在500 ms内停车。

工具每100 ms发送ROS2 M100同款13字节控制帧。连接、断开、松开、鼠标离开、窗口失焦和关闭时都会发送停车帧。
