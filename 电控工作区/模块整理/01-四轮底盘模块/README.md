# 01 四轮底盘模块

## 负责什么

接收ROS2 `/cmd_vel`，控制四个麦克纳姆轮前进、后退、横移和旋转，并通过编码器与PID稳定轮速。

## 数据流

`/cmd_vel` → ROS2 `codbot_base.cpp` → 串口命令码 `0x01` → STM32 `autocontrol.c` → `motor.c` → PID → `bsp_motor.c` → 四路电机

编码器反馈方向：`bsp_encoder.c` → 轮速 → PID/反馈 → ROS2里程计。

## 文件夹

- `STM32代码`：电机、编码器、PID、PWM和共享控制任务的代码副本。
- `ROS2代码`：`/cmd_vel`、串口封包、里程计和启动参数的代码副本。
- `CODE_INDEX.md`：逐个文件说明“看什么、是否共享、源文件在哪里”。

## 安全标记

- `[核心]`：本模块的主要逻辑。
- `[底层]`：直接控制引脚、PWM或读取编码器。
- `[共享]`：同时包含铲斗或系统调度逻辑，不能当成底盘私有文件。
- `[配置]`：修改前必须理解影响范围。

当前PWM上限为300/1000。首次测试必须架空，一次只测试一个轮子。
