# 02 铲斗舵机模块

## 负责什么

使用MG996R和第1路舵机接口控制铲斗的挖豆、运输、卸豆、复位和急停动作。

## 数据流

ROS2 `/bucket_command` → `codbot_base.cpp` → 串口命令码 `0x07` → `autocontrol.c` → `bucket_control.c` → `servoSetPluse()` → PC6/TIM8_CH1

状态反馈方向：STM32命令码 `0x08` → ROS2 `/bucket_state`。

## 文件夹

- `STM32代码`：铲斗状态机、舵机PWM和共享任务代码副本。
- `ROS2代码`：铲斗命令、状态反馈和串口封包代码副本。
- `CODE_INDEX.md`：逐个文件说明“看什么、是否共享、源文件在哪里”。

## 安全标记

- `[核心]`：铲斗动作状态机。
- `[底层]`：直接输出舵机PWM。
- `[共享]`：同时服务底盘、串口或系统任务。
- `[配置]`：标定脉宽与安全开关。

当前 `BUCKET_ENABLE_MOTION=0`，三个位置均为1500μs。完成脱开摇臂的空载标定前，不得改为1。


实物接线前还必须确认舵机插座S/+/−顺序、5V端子正负极，并使用独立5V大电流电源。
