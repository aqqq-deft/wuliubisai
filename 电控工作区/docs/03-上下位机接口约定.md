# 03 上下位机接口约定

## 已有底盘协议

ROS2驱动将 `/cmd_vel` 转为13字节厂家帧：帧头 `CD 0A`、命令码、X方向/速度、Y方向/速度、旋转方向/速度、XOR校验。STM32仅在校验正确时执行。

## 安全行为

- 上电：四轮目标速度和PWM均为0。
- 有效底盘命令：刷新500ms看门狗。
- 超时、校验错误或串口断开：四轮停车并清空PID累计量。
- 蓝牙控制：比赛固件默认关闭。

## 机构命令预留

| 名称 | 建议值 | 含义 |
|---|---:|---|
| STOP | 0 | 停止机构 |
| LOAD_START | 1 | 开始取豆 |
| LOAD_STOP | 2 | 停止取豆 |
| UNLOAD_START | 3 | 开始卸豆 |
| UNLOAD_STOP | 4 | 停止卸豆 |
| RESET | 5 | 机构复位 |
| EMERGENCY_STOP | 255 | 整车急停 |

这些编号目前只是冻结接口设计，尚未占用厂家串口帧命令码；机械接口确定后再加入协议，避免与现有命令冲突。
