# 01 首次上电前检查

## 现在不要做

- 不接电池。
- 不接四个电机。
- 不把不认识的插头强行插入主板。
- 不根据线的颜色猜正负极。

## 请拍摄并提供

- CODBOT-RCU 主板正面全图，文字清晰。
- 主板背面全图。
- 电源输入接口和旁边丝印特写。
- 电池完整标签，必须看清电压、容量和接口。
- TB6612 驱动板正反面。
- 所有下载器和USB小板正反面。
- 四个电机标签以及编码器插头。
- 当前整车全部接线的俯视图。

## 确认表

- [ ] 主控芯片实物确认为 STM32F103ZE 或确认实际型号。
- [ ] 电池标称电压已记录。
- [ ] 电源正负极已通过丝印或万用表确认。
- [ ] 电机额定电压、电流已记录。
- [ ] TB6612 电机电源不超过 13V。
- [ ] 主控板和电机驱动器计划共地。
- [ ] 已确认使用 ST-Link、J-Link 或板载下载方式。
- [ ] 底盘可以稳定架空，四个轮子不会碰地或碰人。

全部完成后才能第一次通电。
