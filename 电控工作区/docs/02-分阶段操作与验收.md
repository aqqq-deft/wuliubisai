# 02 分阶段操作与验收

## A. Keil编译

1. 打开 `firmware/codbot-M100-2022.08.19/Project/MDK-ARM/project.uvprojx`。
2. 点击 Build，不接主板、不点 Download。
3. 验收：0 Error；Warning 需要保留完整文本再判断。

## B. 主板空载测试

前提：电源、极性和下载器已经确认。只接主板，不接电机。

1. 下载固件。
2. 观察LED是否周期闪烁。
3. USB串口使用115200、8N1。
4. 验收：上电无异常发热、无异味，串口能够收发。

## C. 单轮架空测试

1. 底盘架空，只允许低速短时测试。
2. 依次执行停止、正转、停止、反转、停止。
3. 验证编码器方向和速度反馈。
4. 当前PWM被固件限制为30%，未完成单轮验收不得提高。

## D. 四轮动作

按顺序验证：前进、后退、左横移、右横移、原地左转、原地右转。每次只运行1秒并立即停车。若某轮方向错误，记录轮号，不直接乱换多根线。

## E. ROS2

- 包路径：`ros2_ws/src/codbot_base`
- 底盘类型必须为 `M100`。
- 串口115200，默认设备 `/dev/base`。
- `/cmd_vel`：`linear.x`前后、`linear.y`横移、`angular.z`旋转。
- 停止发布命令后，STM32必须在500ms内停车。

## F. 搬运机构

机械组提供电机/舵机型号、额定电压电流、限位数量、动作顺序后才能接线。当前只保留命令：取豆、停止取豆、卸豆、停止卸豆、复位、急停。
