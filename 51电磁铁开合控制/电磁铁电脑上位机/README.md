# 三路电磁铁电脑上位机

这是配套 STC89C52RC 下位机验证程序使用的 Windows 图形控制软件。

## 功能

- M1、M2、M3 三个按钮独立切换通电和断电。
- 支持全部通电、全部断电。
- 每次操作后自动读取下位机状态。
- 显示发送命令和接收回复。
- 主动断开串口或关闭窗口前，自动发送 `ALL OFF`。

## 启动前

1. 开发板中应已经烧录 `MagnetControl51.hex`。
2. CH340 接线：
   - CH340 TXD → 开发板 P3.0/RXD
   - CH340 RXD → 开发板 P3.1/TXD
   - CH340 GND → 开发板 GND
   - CH340 VCC 不接
3. 关闭 SSCOM，并确保 STC-ISP 没有占用同一个串口。
4. 开发板串口参数保持 9600、8N1。

## 使用方法

1. 双击 `启动上位机.bat`。
2. 在串口列表中选择 `USB-SERIAL CH340` 对应的 COM 口，通常是 COM5。
3. 点击“连接”。连接成功后，软件会自动发送 `STATUS`。
4. 点击 M1、M2 或 M3：
   - 灰色表示断电，点击后通电。
   - 绿色表示通电，点击后断电。
5. “全部通电”和“全部断电”可以同时控制三路。
6. 使用结束时先点击“全部断电”，再关闭软件和电源。

## 安全说明

- 主动断开或关闭窗口时，软件会尝试发送 `ALL OFF`。
- 如果 CH340 意外拔出、USB 断开或电脑死机，下位机可能保持断线前的输出状态。此时应立即关闭12V电源。
- 上位机显示的是下位机报告的逻辑状态；首次使用时仍应观察 MOS 指示灯和电磁铁实际动作是否一致。
- 电磁铁不需要长时间通电时，应及时断电，避免持续发热。

## 文件说明

- `magnet_control_gui.pyw`：上位机主程序。
- `启动上位机.bat`：双击启动入口。
- `test_protocol.py`：状态回复解析测试。
